Live broadcasted YouTube videos

Find videos around the world that are currently live broadcasted. Drag the map to explore a different country. You will get all the videos that are live broadcasted around that country within a radius of 500km

This project was developed during a 22h Hackathon at Technical University of Denmark. The source code is available on GitHub.
